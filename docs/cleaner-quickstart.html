cleaner — установка и использование

Очистка iOS-проекта от метаданных машины разработчика перед передачей на сборочную машину.

1. Установка (один раз)

brew tap victorshablyko/cleaner
brew install cleaner
cleaner version   # → cleaner v0.1.0

Обновление: brew update && brew upgrade cleaner

2. На исходной машине

cd /путь/к/Проекту
cleaner export    # ../Проект-clean (копия,
                  # оригинал не трогается)
cd ../Проект-clean
cleaner check     # проверка чистоты
cleaner pack      # ../Проект-clean.zip
                  # с паролем

Передай .zip на ВПС. Ссылку и пароль — разными каналами.

3. На ВПС (сборочная)

cd Проект-clean
cleaner prepare   # xattr -cr + pod install
  • Открыть в Xcode → Clean Build Folder (⇧⌘K)
  • Проверить Signing & Capabilities
  • Product → Archive
  • Залить через Transporter

4. После заливки

cleaner purge     # удалить проект +
                  # DerivedData

Полезные флаги

  • --dry-run — показать без выполнения
  • --keep-pods — не вырезать Pods/
  • --strip-resolved — удалить Package.resolved
  • --git — удалить .git/ (после проверок)
Безопасно: export только копирует — оригинал не изменяется. clean чистит папку на месте (восстанавливаются только файлы под git). Команда работает в любом Xcode-проекте: запускай из его корня.
